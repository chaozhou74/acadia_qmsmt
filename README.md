# acadia_qmsmt  
A Python package for streamlined quantum measurement and data processing.  

The measurement functionality is built on top of the [Acadia](https://git.yale.edu/RSL/acadia) RFSoC control framework.

## Features:
- **YAML-based Parameter Management**: Simplify experiment configuration with human-readable YAML files that directly interface with measurement scripts.  
- **Auto Configuration Functions**: Streamline the creation of new `Runtime` classes with automated setup and helper utilities.
